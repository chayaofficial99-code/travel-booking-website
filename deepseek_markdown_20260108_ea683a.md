# Travel Booking Website

A fully responsive travel booking website built with vanilla HTML, CSS, and JavaScript.

## Features

### 1. Search & Filter System
- Real-time filtering by destination, date, budget, and travel type
- Tab-based search for flights, hotels, and packages
- No page reload required for filtering

### 2. Booking System (Frontend Simulation)
- Flight booking with passenger and class selection
- Hotel booking with room and guest selection
- Tour package booking with duration and package type
- Form validation
- Secure payment UI simulation
- Booking confirmation with details

### 3. User Account System
- Login/Registration with form validation
- Social login UI (Google/Facebook)
- User dashboard with:
  - Booking history
  - Wishlist
  - Saved destinations
  - Travel statistics
- LocalStorage data persistence

### 4. Reviews & Ratings
- Star-based rating UI
- Review submission system
- Average rating display
- Customer reviews listing

### 5. Destination Details
- Popular destinations grid
- High-quality images
- Pricing and ratings
- Quick booking functionality

### 6. Customer Support
- Contact information display
- Live chat simulation
- FAQ section ready

### 7. Multi-language & Multi-currency
- Language switcher (English, Bangla, Arabic)
- Currency converter (USD, BDT, AED)
- Real-time price conversion

### 8. Blog/Travel Guide
- Blog listing page
- Travel tips and guides
- Responsive blog cards

### 9. Social Integration
- Social media share buttons
- Social login UI components
- Social media links

## Technical Features

- **Responsive Design**: Mobile-first approach with responsive breakpoints
- **Semantic HTML5**: Proper use of semantic elements
- **Clean CSS**: Modern CSS with variables and flexbox/grid
- **Vanilla JavaScript**: No frameworks, pure JavaScript
- **Modular Code**: Organized into separate modules
- **LocalStorage**: Persistent data storage for user sessions
- **Form Validation**: Client-side validation for all forms
- **Accessibility**: Keyboard navigation and ARIA labels ready
